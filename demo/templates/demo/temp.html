Information

The iASiS Unified Schema represents the classes, properties and relationships of the iASiS Knowledge Graph.

The iASiS Unified Schema objectives are to model characteristics of heterogenious data integrated in the iASiS Knowledge Graph, and to describe the ontology annotations performed by analysis tools over images, open data, clinical records and genomic data.

Information

semEP is an unsupervised semantics based edge partitioning method; semEP combines a data mining framework for link discovery, semantic knowledge (similarities) from ontologies or semantic spaces, and an algorithmic approach to partition the edges of the iASiS knowledge graph.

Reference
Guillermo Palma, Maria-Esther Vidal, Louiqa Raschid: Drug-Target Interaction Prediction Using Semantic Similarity and Edge Partitioning. Semantic Web Conference (1) 2014: 131-146
